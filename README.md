# 奶龙健身操 🐉💪

一款陪伴式健身桌宠，定时提醒你起身活动，并通过摄像头识别你的肢体动作，跟着奶龙一起跳健身操。

## ✨ 项目亮点

- **奶龙桌宠**：无边框、透明、置顶的 Windows 桌面宠物，安静地趴在屏幕角落
- **定时提醒**：每 8 秒在奶龙头顶弹出可爱气泡，提醒喝水、休息、活动身体
- **肢体识别**：基于 MediaPipe Pose 实时检测人体 33 个关键点（肩、肘、腕、髋、膝、踝等）
- **跟跳健身操**：左侧奶龙领舞视频，右侧实时摄像头 + 骨骼叠加，AI 评分你跟舞的相似度
- **多难度可选**：简单 / 困难两种模式，不同动作库与时长
- **数据沉淀**：本地保存历史成绩、排行榜与个人资料，支持自动备份

## 🏗️ 项目结构

```
.
├── 奶龙桌宠/        # Python 桌宠 (PyQt5)，点击跳转网页健身操
│   └── main.py
└── react-app/       # Web 健身操应用 (React + Vite)
    └── src/
        ├── pages/   # 首页 / 难度 / 运动 / 排行 / 历史 等页面
        ├── utils/   # 姿态识别与评分算法
        └── hooks/   # 本地存储 / 拖拽 / Toast 等
```

## 🚀 快速开始

### 1. 启动桌宠

```bash
cd 奶龙桌宠
pip install -r requirements.txt
python main.py
```

桌宠会出现在屏幕右下角，单击打开 Web 应用，右键退出。

### 2. 启动 Web 应用

```bash
cd react-app
pnpm install
pnpm dev
```

浏览器打开终端输出的地址（默认 http://localhost:5173 ），允许摄像头权限即可开始跟跳。

## 🎮 使用流程

1. 桌宠在桌面安静守候，定时弹出提醒
2. 单击桌宠 → 打开健身操网页
3. 选择难度 → 倒计时 3-2-1 → 跟着奶龙视频开跳
4. 系统通过摄像头识别肢体动作并实时评分
5. 结束后查看成绩、加入排行榜

## 🛠️ 技术栈

- **桌宠**：Python · PyQt5 · Pillow · NumPy
- **Web**：React 19 · Vite 8
- **姿态识别**：MediaPipe Pose
- **包管理**：pnpm
